Forms

Forms are a way to accept and request user input.

Forms are created using the <form></form> tag and can have several different attributes that control what the form should do when a button is clicked.

List of Form Attributes

Forms also can contain different types of elements. The most common being the <input> element. If you notice, the input element does not have a closing tag.

List of Form Elements

Inputs

The <input> element can take a type attribute, which can give you several different types of input options.

Labels

Labels improve accessibility and make forms easier to use, because you can link the labels to other form controls, like checkboxes.

To connect a label to a control element, you use the <for> tag on the label element and the <id> tag on the control element.

Buttons

Buttons help make forms interactive and you create a button using the <button>This is a button</button> tag.

Example Forms

Basic Form with an Input and No Type

Code:

        
            <form>
                <input>
            </form>

Basic Form with an Input and Text Type

Code:

        
            <form>
                <input type="text">
            </form>

Basic Color Picker Form

Code:

        
            <form>
                <input type="color">
            </form>

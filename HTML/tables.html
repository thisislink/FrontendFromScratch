Tables

Important Info About Tables

Tables should only be used to layout actual data that belongs in a table. In the early days of the web, tables were used to create website layouts and themes, which worked, but is no longer the best practice today.

Table Basics

The element represents a single cell in a table.

The element represents a row of cells in a table and groups a row of cells together.

The element represents a header of a group of cells in a table.
